## Core Features:

- Text Input: Accept user-provided text input in any language.
- Language Detection: Automatically detect the input language using AI.
- Translation: Translate the input text into the user's target language (defaulting to English).
- Pronunciation: Provide a text-to-speech feature for translated text, using AI.
- Contextual Tool: Offer an AI tool that detects and provides possible region specific contextual information related to a word from the original text in translation.
- Text Display: Display the original text, translated text, and the detected language in a clear and organized format.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and communication.
- Background color: Very light blue (#E8EAF6).
- Accent color: Teal (#009688) for interactive elements and highlights.
- Body font: 'Inter', a grotesque-style sans-serif for readability and a modern feel.
- Headline font: 'Space Grotesk', a proportional sans-serif that provides a modern feel.
- Use simple, universal icons for language selection and other functions.
- Subtle animations during the translation process to indicate loading.
